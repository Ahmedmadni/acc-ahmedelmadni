# خطة تسريع الموقع

السبب الرئيسي لبطء التحميل: 3 فيديوهات في `public/` بحجم إجمالي ~7MB تُحمَّل دفعة واحدة عند فتح الصفحة الرئيسية، إضافة إلى تحميل خطوط وأيقونات Font Awesome بشكل يحجب العرض، وعدم تقسيم الكود.

## التحسينات المقترحة

### 1. الفيديوهات (الأثر الأكبر)
- جعل فيديو الخلفية الرئيسي (`bg-video.mp4`) Lazy: عدم تحميله إلا بعد تحميل الصفحة (`preload="none"` + تشغيل بعد `load`)، وعرض صورة `poster` (نستخدم `hero-finance-bg.webp`) في البداية.
- الفيديوهات الثانوية (`bg-video-2.mp4`, `bg-video-3.mp4`): `preload="none"` وتحميلها فقط عند الاقتراب من المشاهدة (IntersectionObserver).
- تخفيف الحجم: إضافة سمات `muted playsinline` (موجودة غالبًا) والاكتفاء بفيديو واحد إن أمكن — للنقاش.

### 2. الصور
- إضافة `loading="lazy"` و `decoding="async"` لكل الصور غير المرئية في الـ viewport الأول.
- الصورة البطلة (Hero) فقط تبقى `eager` مع `fetchpriority="high"`.
- preload للصورة البطلة فقط عبر `head().links` في `src/routes/index.tsx`.

### 3. الخطوط والأيقونات
- استبدال Font Awesome CDN (يحمّل CSS كامل + خطوط ضخمة) باستيراد أيقونات `lucide-react` المستخدمة فقط، أو إبقاؤه مع `media="print"` + swap (موجود) لكن إزالة الأيقونات غير المستعملة.
- تقليل أوزان خط Cairo إلى ما هو مستخدم فعليًا (مثلاً 600,700 فقط بدل 400;600;700;800).

### 4. تقسيم الكود (Code Splitting)
- `src/routes/index.tsx` يبلغ 1552 سطرًا ويحتوي كل المكونات (Navbar, Footer, Modals, AIAssistant). نقل المكونات الثقيلة (AIAssistant, الـ Modals, FloatingSocial) إلى ملفات منفصلة لتُحمَّل عند الحاجة فقط.
- التأكد أن `AIAssistant` يُحمَّل ديناميكيًا (lazy) لأنه لا يظهر في الـ viewport الأول.

### 5. الـ Animations
- تقليل عدد عناصر `motion.div` غير الضرورية، واستخدام `viewport={{ once: true }}` على كل `whileInView` لمنع إعادة التشغيل.

### 6. السكربتات الخارجية
- إبقاء Google Tag Manager `async` (موجود) مع تأجيله بعد تفاعل المستخدم إن أمكن.

## الملفات المتأثرة
- `src/routes/index.tsx` — تعديلات الفيديو/الصور/lazy للمكونات
- `src/routes/__root.tsx` — تخفيف أوزان الخط، تأجيل Font Awesome
- ملفات جديدة: `src/components/HeroVideo.tsx`, `src/components/LazyVideo.tsx` (اختياري للتنظيم)

## المتوقع بعد التطبيق
- تقليل حجم التحميل الأولي من ~7MB+ إلى ~500KB.
- تحسّن LCP و TTI بشكل ملحوظ خصوصًا على شبكات الجوال.

هل أبدأ التنفيذ بالكامل، أم تريد البدء بالنقطة الأولى (الفيديوهات) فقط لرؤية الأثر؟
